Quick Start Examples

Getting Started

Clone the repository and install dependencies:

git clone https://github.com/sjhallo07/Crozz-Coin.git cd Crozz-Coin/sui-stack-hello-world/ui pnpm install pnpm dev

The application will be available at http://localhost:5173

Move Smart Contract Example

Simple greeting contract demonstrating object ownership:

module hello_world::greeting { use sui::object::{Self, UID}; use sui::tx_context::{Self, TxContext}; struct Greeting has key, store { id: UID, text: String, } public entry fun create( text: String, ctx: &mut TxContext ) { let greeting = Greeting { id: object::new(ctx), text, }; transfer::transfer(greeting, tx_context::sender(ctx)); } }

React Integration

Connect to Sui network and interact with smart contracts:

import { useSignAndExecuteTransaction } from '@mysten/dapp-kit'; function CreateGreeting() { const { mutate: signAndExecute } = useSignAndExecuteTransaction(); const handleCreate = () => { signAndExecute({ transaction: { kind: 'moveCall', data: { packageObjectId: PACKAGE_ID, module: 'greeting', function: 'create', arguments: ['Hello, Sui!'], } } }); }; return <button onClick={handleCreate}>Create Greeting</button>; }

Deployment Guide

Publish your contract to Sui testnet:

# Publish the Move package cd move/hello-world sui client publish --gas-budget 100000000 # Deploy frontend to GitHub Pages cd ../../ui pnpm build # Upload the dist/ folder to GitHub Pages
