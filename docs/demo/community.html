Frequently Asked Questions

Common questions about CROZZ and Sui development

What is CROZZ?

CROZZ is a demonstration decentralized application built on the Sui blockchain. It showcases how to build full-stack dApps using Move smart contracts and modern web technologies. The project is open-source and designed for learning and experimentation.

What is Sui Testnet?

Sui Testnet is a test environment for the Sui blockchain where developers can deploy and test their applications without using real assets. It's regularly updated with new features and reset periodically. Testnet SUI tokens have no real-world value.

How do I get started?

To get started with CROZZ: 1) Install Sui Wallet browser extension, 2) Get testnet SUI from the faucet, 3) Visit the demo site and connect your wallet, 4) Try creating greetings and interacting with the smart contract. For development, clone the GitHub repository and follow the README instructions.

Can I contribute to CROZZ?

Yes! CROZZ is open-source and welcomes contributions. You can submit bug reports, feature requests, or pull requests on GitHub. Check the CONTRIBUTING.md file in the repository for guidelines on how to contribute.

Where can I get support?

For CROZZ-specific questions, open an issue on GitHub. For general Sui development questions, join the Sui Discord community where thousands of developers are ready to help. You can also check the Sui documentation for comprehensive guides.

Is CROZZ safe to use?

CROZZ is a demo application running on testnet, which means all transactions use test tokens with no real-world value. However, never share your wallet seed phrase or private keys with anyone. For production applications, always conduct thorough security audits before deployment.
